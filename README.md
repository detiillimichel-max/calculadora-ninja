
# 14. Calculadora Ninja

Resolva contas que caem do topo. Errou, perde vida.

## Modos
- **Normal**: +, -, ×, ÷
- **Tabuada**: escolha 2-9 ou Todas (para crianças)

## Loja
- ❤️ Vida = 100 pts
- 🐢 Desacelerar = 50 pts
- 💡 Dica = 30 pts
